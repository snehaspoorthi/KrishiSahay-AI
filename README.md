# 🌾 KrishiSahay

## KrishiSahay: GenAI-Powered Agricultural Query Resolution System

KrishiSahay is an AI-powered platform designed to help farmers access **real-time, personalized agricultural guidance** through text and voice interactions in multiple regional languages.

The system leverages **Large Language Models (LLMs)** and real-time data integrations to provide accurate recommendations on crops, weather, fertilizers, and government schemes.

---

## 🚀 Features

- 🤖 **AI-Powered Query Resolution**
  - Uses LLM (Llama 3 via Groq) to answer agricultural queries  
  - Provides context-aware recommendations  

- 🌐 **Multilingual Support**
  - Supports 8+ Indian languages (Hindi, Telugu, Marathi, etc.)  
  - Custom translation engine for seamless interaction  

- 🎤 **Voice Input & Output**
  - Farmers can ask questions using voice  
  - AI responses are delivered via speech synthesis  

- 🌦️ **Real-Time Weather Integration**
  - Fetches live weather data using OpenWeatherMap API  
  - Helps farmers make informed decisions  

- 📊 **Government Schemes Dashboard**
  - Displays key schemes like PM-Kisan and PMFBY  

- 👥 **Community Forum**
  - Farmers can post queries and interact with others  

- 🛒 **Marketplace (UI)**
  - Interface for buying agricultural inputs and selling produce  

- 🌱 **Soil & Crop Insights (Partial)**
  - UI structure implemented (currently uses placeholder data)
